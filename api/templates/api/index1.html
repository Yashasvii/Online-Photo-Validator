About the project

An Image Processing Based Classifier that detects the diseases in the Chest X-ray. It is a project that aims to develop a model to diagnose chest disease from chest X-Ray. It uses Convolutional Neural Network based model which takes chest X-Ray image as input. On the basis of input, model decides whether the patient is healthy or unhealthy and also determines the probable 14 thorax diseases if unhealthy.

Objective

To study the relevance of data from other countries in the context of NepaL

Scope

Supportive Technology to the current manual method of diagnosis.
